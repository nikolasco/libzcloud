Download Consumer¶

class DownloadConsumer¶
write(buffer, bytes, error)¶
Parameters:
  • buffer – storage containing the bytes that shuld be consumed
  • bytes – the number of bytes available for consumption
  • error – optional storage for an error
Returns:

the number of bytes actually read from buffer
reset(error)¶

Reset the consumer. All previous calls to write should be “forgotten”.

Parameters:
  • error – optional storage for an error
Returns:

true on success and false on error
