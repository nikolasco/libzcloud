Upload Producers¶

A upload producer is an object which provides data received for a store to upload. Any subclass of UploadProducer will work with any ZCloudStore instance, but libzcloud provides a few common producers which cover most uses.

Memory Upload Producer¶

This consumer reads a fixed-length, preallocated buffer.

ZCloudMemoryUploadProducer *zcloud_memory_upload_procuer(gpointer buffer, gsize buffer_length)¶
Parameters:
  • buffer – the buffer to read data from.
Returns:

an upload producer

FD Upload Producer¶

This consumer simply reads a file descriptor, which the caller is responsible for opening and closing. The file descriptor must be blocking. It may or may not be seekable. If it isn’t, reset and calculate_md5 will return an error; this implies that uploads can not be retried or the integrity verified.

ZCloudFDUploadProducer *zcloud_fd_upload_procuer(int fd)¶
Parameters:
  • fd – the file descriptor to use
Returns:

an upload producer
